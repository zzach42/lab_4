# lab_4
RightTriangle

Вычисление площади прямоугольного треугольника.
Даны длины катетов a и b типа double.
Результат вычисления площади должен быть типа double.

Сделаем класс RightTriangle с двумя полями a и b, которые представляют длины катетов прямоугольного треугольника.
У класса реализуем три конструктора:

без параметров,

с двумя параметрами (a, b),

с одним параметром (a), при этом второй катет приравнивается первому.

Далее реализуем метод area(), который вычисляет площадь прямоугольного треугольника по формуле
S = 0.5 * a * b.
Если треугольник не существует (катеты меньше либо равны нулю), метод возвращает отрицательное число.

Также добавим метод isValid(), который проверяет существование треугольника (оба катета должны быть положительными).

Далее реализуем функцию create(), которая позволяет создать экземпляр класса с клавиатуры.
Функция выполняет проверку корректности ввода: если пользователь вводит некорректные данные (отрицательные числа или ошибку ввода), создаётся пустой объект и выводится сообщение об ошибке.

После этого перегрузим унарные операторы ++ и --.
Оператор ++ увеличивает длины катетов в 2 раза, а оператор -- уменьшает длины катетов в 2 раза.
Сначала реализуется префиксная версия, а постфиксная реализуется через неё.

Далее реализуем операции приведения типов:

явное приведение к double — возвращает площадь треугольника, если он существует, и отрицательное число в противном случае;

неявное приведение к bool — возвращает true, если треугольник существует, и false в противном случае.

Затем перегрузим бинарные операторы <= и >=, которые сравнивают площади двух треугольников.

Также переопределим оператор вывода <<, чтобы при выводе экземпляра класса в поток выводились значения катетов треугольника.
